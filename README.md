# 買物物語

## ゲーム概要
敵の攻撃をギリギリで回避することで生まれるスリルにフォーカスした、
買物袋を動物に盗まれないようにしながら帰宅を目指す横スクロールアクションゲームです。

プレイヤーは攻撃を避けながら進み、
所持している商品を守りつつゴールを目指します。

## 制作時に目指したこと・意識したこと

### ・回避によるスリルの最大化
敵の攻撃を「ギリギリで避ける」体験を重視し、
プレイヤーに緊張感と達成感を与えるゲームデザインを目指しました。

### ・部位ごとの当たり判定によるゲーム性の拡張
攻撃を受けた部位によって失う商品を変化させることで、
単純な被弾ではなく、状況に応じたリスク管理が生まれる設計を意識しました。

### ・追従型の敵による圧力設計
スーパーマリオブラザーズ3に登場する太陽のように、
プレイヤーを追い詰める敵を実装し、
常にプレッシャーがかかる状況を作ることを目指しました。

---

## そのために行ったこと

### ① 部位ごとの当たり判定の実装
キャラクターに複数の矩形当たり判定を設定し、
各部位ごとに異なる処理を紐付けました。  
これにより、被弾位置に応じたゲーム性の変化を実現しています。

### ② 商品システムの設計
各当たり判定に対応する形で、
奪われる商品をランダムで追加・管理する仕組みを実装しました。  
これにより、プレイごとに状況が変化するリプレイ性を持たせています。

### ③ サインカーブを用いた敵の動きの実装
y = sinθ の動きをベースにした軌道を採用し、
敵キャラクター（鳩）の移動に緩急を持たせました。  

- 上昇開始  
- 頂点付近の減速  
- 降下時の加速  

といった変化を加えることで、
自然かつプレイヤーにプレッシャーを与える動きを実現しています。
